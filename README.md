# Building

## MacOS/Linux

### Dependencies
* GCC
* make

### Building

Simply run ```make``` to build.

### Uninstalling

Just run ```./uninstall.sh``` or ```bash uninstall.sh``` or ```sh uninstall.sh``` or something.

## Windows

### Dependencies
* GCC

### Building

Run ```make.bat``` to build.

### Uninstalling

Run ```uninstall.bat``` to uninstall.
